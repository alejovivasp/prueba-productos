  • Para desplegar la aplicación debe descomprimir el contenido en un web server de forma que se puede acceder a todos los archivos en las carpetas
  • Para cambiar la URL con los productos y categorias editar la linea 8 del archivo js/app/ProductosCategorias.js con el valor:
    productos.modelo.URL_PRODUCTOS = 'data.json';
    Por:
    productos.modelo.URL_PRODUCTOS = 'http://servidor/ruta/';
    donde http://servidor/ruta/ corresponde a la nueva URL.
  • La URL que se configure debe estar en el mismo servidor, de lo contrario el navegador podrá bloquear la petición indicando un error "cross-origin resource sharing" (CORS)
  • La URL debe regresar el contenido mediante una peticion http usando el metodo GET
  • El tiempo de carga de las imágenes es lento porque se evito el cache del navegador para que se desplegaran imágenes distintas
  • Para ver la compatibilidad de navegadores ir http://foundation.zurb.com/sites/docs/compatibility.html
